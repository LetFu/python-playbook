# Python Notebooks Playground

A collection of interactive Jupyter notebooks for learning Python programming.

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/LetFu/python-playbook.git
   cd python-playbook
   ```

2. (Optional) Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Start Jupyter:
   ```bash
   jupyter notebook
   ```

5. Open the `notebooks/` folder and start learning!

## Notebooks

| Notebook | Description |
|----------|-------------|
| [01_getting_started.ipynb](notebooks/01_getting_started.ipynb) | Variables, data types, basic operations, control flow, and loops |
| [02_data_structures.ipynb](notebooks/02_data_structures.ipynb) | Lists, tuples, dictionaries, and sets |
| [03_functions_and_classes.ipynb](notebooks/03_functions_and_classes.ipynb) | Functions, classes, and object-oriented programming |
